# PAW - Purr-Accessible Webnames

A DNS alternative that makes the web more accessible.

## Overview

PAW (Purr-Accessible Webnames) is a DNS alternative protocol designed to provide an accessible naming system for web resources.

### Technical Specifications

- **Primary Protocol**: TCP on port 96
- **Future Support**: UDP on port 96 (planned)
- **Primary Language**: Swift

## Project Structure

This repository serves as a monorepo containing multiple components of the PAW ecosystem.
